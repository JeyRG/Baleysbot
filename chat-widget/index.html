Chat Agent Widget

Uno de los widgets de chat más elegantes y fáciles de integrar. Diseñado para ser copiado y usado en cualquier proyecto web en segundos.

¿Cómo usarlo en otro proyecto?

1. Copia la carpeta

Copia el directorio chat-widget completo a tu nuevo proyecto.

2. Enlaza los archivos

Añade el CSS y el JS a tu HTML principal antes de cerrar el body.

3. Inicializa

Crea una instancia de ChatAgent con tus preferencias.

Código de Ejemplo

<!-- 1. Hoja de estilos -->
<link rel="stylesheet" href="chat-widget/widget.css">

<!-- 2. Script de lógica -->
<script src="chat-widget/widget.js"></script>

<!-- 3. Inicialización -->
<script>
  const bot = new ChatAgent({
    agentName: "Baleys Bot",
    welcomeMessage: "Hola, ¿cómo puedo ayudarte?",
    primaryColor: "#10b981"
  });
</script>
